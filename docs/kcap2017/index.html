Fuzzy Semantic Labeling for Semi-structured Numerical Data Sources
(Additional Material)

KCAP 2017 (under submission)

Authors: Ahmad Alobaid (aalobaid at fi.upm.es), Oscar Corcho (ocorcho at fi.upm.es)

Abstract

SPARQL endpoints and Linked Data are rich sources of data on the Web that can be potentially used to classify other less structured data sources. In this paper, we propose an approach to suggest types for the numerical columns of a collection of input files available as CSVs. The approach is based on the application of the fuzzy c-means clustering technique to numerical data in the input files, using data from existing SPARQL endpoints in order to train the model. We evaluate our approach against manually annotated input files in the domain of Olympic Games. The results show that the proposed approach classifies most of the types correctly for our test set in that domain. Our approach has the advantages of being domain agnostic (it can use any SPARQL endpoint as an input) and avoiding tedious and costly manual training to match values with types.
